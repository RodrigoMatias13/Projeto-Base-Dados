# Projeto de Base de Dados – VINISYS

Este repositório contém o projeto desenvolvido no âmbito da disciplina de **Base de Dados**, cujo objetivo é a construção de um sistema de informação para a gestão de uma empresa distribuidora de vinhos.

O sistema, denominado **VINISYS**, permite gerir toda a informação relacionada com:

* Produção de vinhos
* Colheitas e vinhas
* Produtores e colaboradores
* Clientes e vendas

---

## Estrutura do Projeto

* `PARTE1/` – Modelação e implementação da base de dados
* `PARTE2/` – Automatismos, queries e aplicação web

---

## Descrição das Partes

### Parte 1 – Modelação e Implementação

Nesta fase foi desenvolvido o modelo da base de dados com base nos requisitos do sistema VINISYS.

Inclui:

* Modelação conceptual e lógica (BoUML)
* Definição das entidades e relacionamentos
* Scripts SQL para:

  * Criação de tabelas
  * Inserção de dados
* Relatório descritivo do sistema

O sistema modela informação como:

* Vinhos, colheitas e castas
* Vinhas e regiões
* Produtores e colaboradores
* Clientes, vendas e devoluções

---

### Parte 2 – Automatismos, Queries e Aplicação Web

Nesta fase foi desenvolvida a componente prática sobre a base de dados fornecida.

Inclui:

#### Automatismos

* Triggers
* Stored Procedures
* Funções

#### Queries e Views

* Consultas SQL para análise de dados
* Criação de views para simplificação de acesso à informação

#### Aplicação Web

* Desenvolvimento de um protótipo em **PHP**
* Funcionalidades:

  * Registo de colheitas
  * Listagem e pesquisa de vinhos
  * Consulta detalhada de vinhos
  * Gestão de clientes e vendas

---

## Tecnologias Utilizadas

* SQL
* PHP
* Sistema de Gestão de Base de Dados (ex.: MySQL)
* BoUML (modelação)
* HTML (interface web)

---

## Como Executar

1. Importar os scripts SQL para o SGBD
2. Configurar ambiente web (XAMPP, WAMP ou LAMP)
3. Colocar os ficheiros PHP no servidor
4. Aceder à aplicação através do navegador
