# Parte 2 – Automatismos, Queries e Aplicação Web

Esta parte do projeto desenvolve funcionalidades avançadas sobre a base de dados VINISYS, incluindo automatismos, consultas e uma aplicação web.

---

## Objetivo

Implementar mecanismos de automatização e análise de dados, bem como um protótipo web para interação com a base de dados.

---

## Conteúdo

* `sql/` – Scripts com:

  * Triggers
  * Stored Procedures
  * Funções
  * Queries e Views

* `php/` – Código da aplicação web

* `imagens/` – Capturas de ecrã da aplicação

* `video/` – Demonstração do sistema

---

## Automatismos

### Triggers

* Atualização automática de dados ao inserir registos
* Gestão de eventos na base de dados

### Stored Procedures

* Registo de participações e colheitas
* Processamento de dados com lógica reutilizável

### Funções

* Cálculo de métricas como:

  * Total de horas trabalhadas
  * Área cultivada

---

## Queries e Views

* Consultas para análise de dados:

  * Vinhos mais premiados
  * Castas mais produzidas
  * Vendas e devoluções

* Views para simplificação:

  * Informação de vinhos e produtores
  * Empresas com maiores compras

---

## Aplicação Web

Desenvolvida em **PHP**, permite:

* Registo de colheitas
* Listagem e pesquisa de vinhos
* Consulta detalhada de vinhos
* Gestão de clientes
* Análise de vendas

---

## Como Executar

1. Importar os scripts SQL
2. Configurar servidor web (XAMPP/WAMP/LAMP)
3. Colocar os ficheiros PHP no servidor
4. Aceder via navegador

---

## Observações

* Utiliza uma base de dados fornecida pela equipa docente
* Integra várias tecnologias (SQL + PHP)
* Demonstra aplicação prática dos conceitos de base de dados
